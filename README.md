## COMO FAZER COMMITS
Se escreve no imperativo, EXEMPLO: feat: método de validação de CPF
- feat: → nova funcionalidade
- fix: → correção de bug
- docs: → documentação
- refactor: → refatoração
- test: → testes
- chore: → tarefas de manutenção (como .gitignore, configs, etc.)
